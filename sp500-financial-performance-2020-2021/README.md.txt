# S&P 500 Financial Performance (2020-2021)

**Tools:** SQL Server, Power BI (DAX), R (ARIMA).
**Goals:** Analyze how S&P 500 companies recovered from 2020 to 2021 - growth, profitability, and top performers.

## Contents
- `/sql/` - data checks, cleaning, and summary queries
- `/powerbi/` - .pbix report (dashboards)
- `/r/` - ARIMA forecast script
- `/images/` - dashboard screenshots
- `/docs/` - project write-up (PDF)

## Highlights
- Total Revenue rose from ~5.6T (2020) to ~13.1T (2021)
- Net Income Margin improved from 8% -> 11%
- Standouts: Apple (profitability), Walmart/Amazon (scale), Moderna/Regeneron (growth) 

## How to use
Open the `.pbix` in Power BI Desktop. See `/r` for forecasting code.
